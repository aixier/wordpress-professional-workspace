## Translation

The theme includes translation-ready strings and supports:
- Chinese (Simplified) - zh_CN
- English - en_US

To add translations:
1. Use tools like Poedit or Loco Translate plugin
2. Translate the strings in `/languages/` directory
3. Upload .po and .mo files
