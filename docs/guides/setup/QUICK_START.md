# 🚀 CardPlanet 快速启动指南

## 1. 启动WordPress
```bash
docker-compose -f docker-compose-simple.yml up -d
```

## 2. 安装ACF插件
1. 访问：http://localhost:8080/wp-admin
2. 登录：petron / Petron12345^
3. 插件 → 安装插件 → 搜索 "Advanced Custom Fields"
4. 安装并启用

## 3. 查看效果
访问：http://localhost:8080

## 完成！
CardPlanet主题已激活，ACF插件已安装，可以开始使用！
